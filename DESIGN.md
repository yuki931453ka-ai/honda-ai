# Honda Satoru AI — DESIGN.md

## コンセプト

**"温もりのある対話"** — おもてなしの心を感じるメンターとの語らい

本田覚さんは15年以上チームエリートを維持し続ける実績を持ちながら、
被災地支援に何度も足を運ぶ温かい人柄。
「成幸家族を増やす」という理念に表れる通り、
派手さではなく"人のぬくもり"を大切にする人物像。

デザインは **Warm Hospitality（温かいおもてなし）** を軸に、
日本的な奥ゆかしさと、Pleasureの"喜び"を融合する。

## 参考デザイン

| ソース | 取り入れる要素 |
|--------|---------------|
| Starbucks | Warm Cream canvas (#f2f0eb)、Gold accent (#cba258)、café的な安心感 |
| Airbnb | 写真的温かみ、柔らかい角丸、人間中心のUI |
| Apple | 余白の美、タイポグラフィの抑制、ノイズの排除 |
| 和の美学 | 侘び寂び的な素材感、自然な色彩、縦書き的リズム |

## カラーパレット

```
--kinari:       #f5f1ea    /* 生成 — メインキャンバス（和紙的な温かみ） */
--surface:      #fffefb    /* 白磁 — カード・バブル背景 */
--surface-warm: #f0ebe2    /* 砂色 — サブ面 */
--ink:          #2c2825    /* 墨 — メインテキスト */
--ink-soft:     #6b6560    /* 灰墨 — セカンダリテキスト */
--konjiki:      #b08d57    /* 金色（こんじき）— プライマリアクセント */
--konjiki-soft: rgba(176,141,87,.10)  /* 金色の光 */

```

## タイポグラフィ

| 用途 | フォント | サイズ | Weight |
|------|---------|-------|--------|
| 見出し | "Noto Serif JP", serif | 20-24px | 500 |
| 本文 | "Noto Sans JP", sans-serif | 14-15px | 400 |
| キャプション | "Noto Sans JP" | 12px | 400 |
| ボタン | "Noto Sans JP" | 14px | 500 |

Google Fonts CDN: `Noto+Sans+JP:wght@400;500&Noto+Serif+JP:wght@500`

## スペーシング

Base: 4px  
Scale: 4 / 8 / 12 / 16 / 20 / 24 / 32 / 48 / 64

## 角丸

| 要素 | Radius |
|------|--------|
| メッセージバブル | 16px（角のない柔らかさ） |
| カード | 12px |
| ボタン | 24px（pill形状） |
| アバター | 50%（円） |
| 入力欄 | 16px |

## 影

控えめな1層シャドウ。重さを出さず、浮遊感を演出。

```
--shadow-sm:  0 1px 2px rgba(44,40,37,.06)
--shadow-md:  0 2px 8px rgba(44,40,37,.08)
--shadow-lg:  0 4px 16px rgba(44,40,37,.10)
```

## アバター

- 金色（--konjiki）の円に白文字の漢字「覚」
- サイズ: ヘッダー 48px / メッセージ内 32px / ウェルカム 80px
- フォント: Noto Serif JP（品格）

## メッセージバブル

- **AI側:** --ai-bubble 背景 + --shadow-sm + 左寄せ
- **ユーザー側:** --konjiki 背景 + 白文字 + 右寄せ
- パディング: 14px 18px
- 最大幅: 680px
- 行間: 1.8（日本語の読みやすさ重視）

## ヘッダー

- 背景: --surface にごく薄い下ボーダー
- アバター + 名前 + サブテキスト
- 名前: Noto Serif JP で品格を出す
- 固定表示（sticky）

## ウェルカム画面

- 大きなアバター（80px）
- Noto Serif JP の挨拶文
- サジェストチップは --surface-warm 背景の pill
- 全体的に余白を大きくとる（呼吸感）

## モーション

- フェードイン: 0.3s ease-out（急がない）
- タイピングインジケーター: 3ドットの柔らかいバウンス
- ホバー: border-color の遷移 0.2s
- active: 控えめな scale(0.98)

## トーン

| ✓ する | ✗ しない |
|--------|---------|
| 温かい | 冷たい・テック感 |

| 品格ある抑制 | 派手・ギラギラ |
| 日本的な奥ゆかしさ | 過度な装飾 |
